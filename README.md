viccuad's Spacemacs configuration
==============================

install
-------

Install spacemacs:
```
~$ git clone https://github.com/syl20bnr/spacemacs ~/.emacs.d
```

install these configs:
```
~$ git clone git@gitlab.com:viccuad/spacemacs.d.git .spacemacs.d
```

test
----

```
SPACEMACSDIR=~/code/emacs.d emacs
```
